# BCOG200 Pacman Game

For my final project, I decided to create a fun game of Pacman. I might add a fun little twist, but my basic idea is based on the Pacman game.

## Planned Functions

1. 'move_pacman`
   This function will be used to move the Pacman character based on what the player inputs.

2. 'move_ghost`
   This function will be used to move the ghosts at random.

3. 'eat_pellet`
   This function will be used to increase the player's score and remove a pellet from the game.
